# Gitea: Homebrew

[![Join the chat at https://gitter.im/kleister/kleister](https://badges.gitter.im/Join%20Chat.svg)](https://gitter.im/kleister/kleister)

Homebrew repository to install Gitea on macOS.

## Prepare

```bash
brew tap go-gitea/gitea
```

## Install

### gitea

```bash
brew install gitea
gitea -h
```
